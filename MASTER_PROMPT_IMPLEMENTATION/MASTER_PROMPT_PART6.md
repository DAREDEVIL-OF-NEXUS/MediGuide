PHASE 6 OBJECTIVE

The current Medicine Library is not medically authoritative.

It currently stores Gemini-generated descriptions, side effects, contraindications and interactions.

That is acceptable for an MVP, but not acceptable according to the final architecture we agreed upon.

Replace the current LLM-generated medical knowledge with a real medical database.

The chatbot should answer using verified medical information whenever possible.

Gemini should become the reasoning layer, not the source of medical facts.
